# n8n-scrapper

What This Workflow Does
This automated workflow creates a professional CRO (Conversion Rate Optimization) analysis service that:

Accepts a landing page URL through a web form
Scrapes the entire landing page content automatically
Analyzes the page using Google Gemini AI with expert CRO insights
Generates a detailed report with:

A candid, engaging "roast" of the landing page
10 specific, actionable CRO recommendations


Creates a Google Doc with the complete analysis, ready to share with clients or team members

The workflow transforms any landing page URL into a professional CRO audit document in minutes, eliminating hours of manual analysis work.
